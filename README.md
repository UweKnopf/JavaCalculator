A Java Calculator using Swing and MVC.

**Basic Requirements**

- Display
- 10 Num Keys
- 4 Operators (+, -, x, /)
- Calculate (=)
- Clear Function

It should be possible to chain calculations (e.g. 2+2+2...) and safeguards against common math errors (e.g. dividing by 0).
